Real-Time Hardware Accelerated Raytracing in Vulkan

This personal project is the result of my interest in high speed raytracing techniques and the Vulkan API. By utilizing Vulkan’s hardware accelerated raytracing I was able to implement realistic features like shadows, reflections and transparency all in real-time. Coming together for a level of visual realism that I am very proud of!

Dynamically Deformable Heightmap via Tessellation Shaders in Vulkan

These heightmaps are my favorite personal contribution to the NiteLite Engine, a custom 3D engine rendered using Vulkan. These heightmaps were designed to be able to deformed during runtime without affecting the performance. In addition the geometry of the heightmap's mesh is subdivided on the GPU using tessellation shaders. This was done to avoid sending large amounts of data while maintaining a high resolution final result.

Custom 2D Shadow Casting via Compute Shaders in OpenGL

This shadow casting algorithm was one of my personal contributions to KnightLight, a platformer game made in a custom 2D engine. This shadow casting algorithm, inspired by 3D raytracing, uses compute shaders to efficiently cast shadows while accurately lighting the terrain itself. Switching to this algorithm from more common approaches not only allowed the system to meet design requirements it also greatly improved the engine’s performance.
