Overview:

This code provides an abstraction layer for OpenGL compute shaders, allowing for their creation, management and execution. This implementation manages the compute shader's resources (buffers, textures and uniforms) automatically, allowing for easy shader creation and utilization. The core features include the ability to load shader code from files, compile and link shaders, dispatch compute tasks, handle GPU memory and data binding.
